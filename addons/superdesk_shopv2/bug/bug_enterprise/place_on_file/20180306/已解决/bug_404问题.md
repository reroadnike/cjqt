


零晨时遇到一起劫持事件,在pc上会显示err_connection_reset,
在手机上显示404....用的是一个深圳不限量的手机卡上的网...
带ssl的网站是没问题的..所以..我认为..全站https是有必要的

晚上,发现主要还不是这个问题,
而是内存不足的问题

内存不足导致
前端:拿不到授权缓存,必定会请求微信
请求微信授权返回,又没内存空间存放,异常转去404|502页面,而404现在没有默认页面
后台:后端只有很小地方用到,所以后台是能正常使用的

处理
1.清了一些命中很低的缓存(大概有1G内存,现在服务器空闲大概就1G,当服务器可用内存小于100M时,前端会有404问题),
2.再看一下京东消息处理内存的消耗
